Angular Guild Quest

This is the Angular version of my Guild Quest project

The current project is using .Net Core 2.1 and Angular 6.

This project is still a work in progress and has not been optimized for mobile platforms.
It is best viewed from a laptop or desktop.

Features

ASP.Net Core 2.1 API
Entity Framework Core 2.1
Angular 6
Bootstrap 4.0
HTML5/CSS3/SASS
SQL Server 2016
Linq
Twilio - used for two party authenication
Microsoft Identity
CKEditor - inline text editor
